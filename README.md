# Assignment 47 - Medical Image Classification

## Repository Link

🔗 [Remote Repository](https://github.com/MrGladiator14/assignment47)

## Project Overview

This project focuses on medical image classification using deep learning techniques. It includes implementations for chest X-ray image analysis and medical imaging metadata processing.

## Project Structure

```
assignment47/
├── .git/                     # Git version control
├── .venv/                    # Python virtual environment
├── Image/                    # Image datasets
│   ├── chest_xray_images.npy # Chest X-ray image data
│   └── chest_xray_labels.csv # Corresponding labels
├── img_cls.ipynb            # Image classification notebook
├── medical_img.ipynb        # Medical imaging analysis notebook
├── medical_imaging_meta.csv # Medical imaging metadata
├── pyproject.toml           # Project dependencies and configuration
├── uv.lock                  # Dependency lock file
└── README.md                # This file
```

## Notebooks

- **img_cls.ipynb**: General image classification implementations
- **medical_img.ipynb**: Medical imaging specific analysis and models

## Data

- Chest X-ray images stored as NumPy array
- Labels and metadata available in CSV format
- Medical imaging metadata for additional analysis
